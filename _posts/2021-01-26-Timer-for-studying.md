#### Notes from Python

Some basic differences emerge to Java - in Python you use // with integer division and / with float division while in Java you only use /. Also raising to power happens differently in Python (**) than in Java pow(). Another "big" difference comes with ++ and -- which are not supported in Python. When I do loops in Python I have to indent the code block, no semicolons needed. 

Difference in if-else statement in comparison to Java comes in the elif. In Java you have if, else if and else, whereas in Python they are if, elif and else. Blocks are indented also here. Breaking the loop is similar in both languages - just use statement break.  

Interesting thing with functions is the docstring that one can access with command help(function_name). It is also good to keep in mind the difference between name arguments and positional arguments in functions. With these two functionalities comes also packing and unpacking of arguments (* for positional, ** for named)
