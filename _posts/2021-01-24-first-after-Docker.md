## Thoughts after Docker 

I just finished the Devops with Docker Mooc course from Helsingin Yliopisto. I must say that I enjoyed it tremendously. There were a lot of useful concepts and tools that I can use later on. Of course the whole containerization and ability to move the images and containers from one machine to another and run them without problems is a **huge** thing but in addition the course taught how to create pipeline through Github and how to deploy to Heroku. So lots of nice things! Bash scripting had to be utilised so it helps a lot if you have already some background in Linux and Bash.  
  
I can highly recommend the course for everyone. Of course now the problem for me is what to select next. I am still pondering if I should choose Devops with Kubernetes (would go nicely after Docker) but as that course ends already in February there is not enough time to really concentrate on that. So I think I'll go with Data Analysis with Python and Cyber Security Base which both are also moocs from University of Helsinki. 
  
And what is best is that all these courses are also free.
