So, after pondering a while I decided to advance with Python and Cyber security base instead of Kubernetes or Full Stack. The reason was partly that at the moment there are issues regarding cyber security so the topics on that course are really relevant for me. On the other hand, having seen some of those hacking videos on youtube has also raised my interest on learning about that. 

#### Getting things started

First impression about this [course](https://csmastersuh.github.io/data_analysis_with_python_2020/index.html) was that do I really have to install all this stuff? Well, luckily I already had VS Code installed and got the TMC plugin up and running quite quickly. After that the problems only started. I was struggling a lot to get the TMC tests working. Then after reading the material found this useful command CTRL+Shift+A that listed all the TMC commands. Then from the list it was easy to spot CTRL+Shift+T to test. Submission window opened after testing and from there on it was plain and simple. 

The first part of the course is going through Python basics. I am about to go that through and then move to the [cyber security base](https://cybersecuritybase.mooc.fi/) as it requires Python for the exercises. Lately my programming has been mostly Java so it is nice to get a refresh from another language. Python is also so flexible and used so widely so I don't think there is nothing against learning that as well. I also think that it gives a nice spice for the bash scripting. 

#### Scheduling

We'll see how this will progress! I am going to allocate 1 hour for workdays, then a bit more for weekends for learning new stuff. With Docker course that was a schedule that was somewhat easy to maintain, not too much but not too little.
